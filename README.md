UC IT 4015 Lab 7 Assignment: Intro to Python for Data Analysis:

Task 1: Your First Python Script

    Goal: Write and execute a simple script that prints "Hello, Python World!" to the screen.
    Instructions: Open your Python IDE or Jupyter Notebook and type print("Hello, Python World!"). Execute the script and observe the output.

 
Task 2: Variables and Data Types

    Goal: Learn about different data types in Python and how to use variables.
    Instructions:
        Create variables to store your name (as a string), your age (as an integer), and your height in meters (as a float).
        Use the print() function to display each variable.

 
Task 3: Lists and Dictionaries

    Goal: Explore Python's list and dictionary data structures.
    Instructions:
        Create a list called favorite_foods with at least three items and print it.
        Create a dictionary called book_info containing keys for title, author, and year and their respective values. Print the dictionary.

 
Task 4: Control Flow

    Goal: Understand how to control the flow of your Python scripts using conditional statements and loops.
    Instructions:
        Write a simple if statement that checks if a variable number is positive, negative, or zero and prints an appropriate message for each case.
        Use a for loop to print each item in your favorite_foods list.

 
Task 5: Functions

    Goal: Learn to write reusable pieces of code with functions.
    Instructions:
        Define a function named greet that accepts a name and prints "Hello" followed by the name. For example, greet("Alice") should print "Hello Alice".
        Call your function with a few different names.

 
Task 6: Combining Concepts in a Script

    Goal: Use what you've learned to write a small script.
    Instructions:
        Write a script that asks the user for their name and age, stores the values in variables, and then prints a message greeting them by name and tells them how old they will be in 5 years.
